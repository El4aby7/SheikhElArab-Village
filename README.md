# Abu Khaled Village Website

This project implements the website for **Abu Khaled Village**, a Middle Eastern dining experience. The website is built using HTML, Tailwind CSS, and JavaScript.

## Features Implemented

1.  **Branding & Currency**:
    *   Updated the site branding to **Abu Khaled Village** across all pages.
    *   Changed all pricing currency to **L.E** (Egyptian Pounds).

2.  **Theme Customization**:
    *   Added a **Dark/Light Mode** toggle button.
    *   The user's theme preference is saved in local storage.

3.  **Localization**:
    *   Added an **Arabic Language** toggle button.
    *   Implemented RTL (Right-to-Left) layout support for Arabic.
    *   Dynamic text translation for key content.

## Project Structure

*   `index.html`: The main landing page.
*   `menu.html`: The detailed menu page.
*   `reservations.html`: The reservations and booking page.
*   `js/main.js`: Handles theme switching, language translation, and general interactivity.
